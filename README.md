**Movie Search & Favorites App**

A responsive, feature-rich movie search application that allows users to explore movies, apply filters, view detailed information, and manage a persistent favorites list. Built with vanilla JavaScript and powered by the OMDB API.

**App Demo**
https://amooayomikun.github.io/JavaScript-Movie-Finder-App/

Overview

MovieFinder is a responsive web application built with vanilla JavaScript that enables users to:

Search for movies in real-time

Filter results by year and type

Sort movies by title or year

View detailed movie information in a modal

Add or remove movies from a favorites list

Persist favorites using browser local storage

The project focuses on clean UI, responsive design, and interactive user experience without relying on frameworks.

**Features**
**Search & Discovery**
  •	Real-time movie search using OMDb API 
  •	Debounced input for optimized API calls 
  •	Error handling for empty or invalid searches 
**Filtering & Sorting**
  •	Filter by: 
    o	Year 
    o	Type (Movie, Series, Episode) 
  •	Sort results by: 
    o	Title (A–Z) 
    o	Year (Newest first) 
**Favorites System**
  •	Add/remove movies from favorites 
  •	Persistent storage using localStorage 
  •	Instant UI updates when toggling favorites 
  •	Favorites count display 
**Movie Details Modal**
  •	View detailed movie information: 
    o	Poster 
    o	Rating 
    o	Genre 
    o	Runtime 
    o	Director & Cast 
    o	Plot summary 
•	Add/remove from favorites directly inside modal 
**Pagination**
  •	“Load More” functionality for additional results 
  •	Smooth incremental data loading 
**Responsive Design**
  •	Fully responsive layout for desktop, tablet, and mobile 
  •	Adaptive grid system 
  •	Mobile-friendly interactions
