**Movie Search & Favorites App**

A responsive, feature-rich movie search application that allows users to explore movies, apply filters, view detailed information, and manage a persistent favorites list. Built with vanilla JavaScript and powered by the OMDB API.

**App Demo**

https://amooayomikun.github.io/JavaScript-Movie-Finder-App/

**Overview**

MovieFinder is a responsive web application built with vanilla JavaScript that enables users to:

  Search for movies in real-time
  
  Filter results by year and type
  
  Sort movies by title or year
  
  View detailed movie information in a modal
  
  Add or remove movies from a favorites list
  
  Persist favorites using browser local storage

The project focuses on clean UI, responsive design, and interactive user experience without relying on frameworks.

**Features**

**• Live Search (Debounced):** Search results update automatically as you type, optimized with a 500ms delay to minimize API calls.

**• Advanced Filtering:** Filter by Release Year and Media Type (Movie, Series, Episode) with dynamic "Active Filter" chips.

**• Dynamic Pagination:** Load More functionality to fetch subsequent pages of results seamlessly.

**• Favorites System:** One-click favoriting directly from the search grid.

  •	Persistent storage using local Storage.
  
  •	Dedicated Favorites Gallery with remove capabilities.
  
**• Detailed Modals:** View expanded movie data including Plot, Director, Actors, and IMDb Ratings.

**• Responsive Design:** Fully optimized for Mobile, Tablet, and Desktop views.

**How It Works**
1.	User enters a movie title

2.	App sends a request to OMDb API
   
3.	Results are displayed dynamically

4.	User can:
  •	Filter results
  •	Sort data
  •	View details
  •	Save favorites
  
Favorites are stored locally using local storage.
